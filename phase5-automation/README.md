# ⚙️ Phase 5 — Automation (n8n + Slack)

## 🚀 Overview

This phase implements automated alert workflows using n8n to reduce manual security monitoring effort.

The system connects AI-based threat detection with real-time alerting and response.

---

## 🧠 Architecture

Logs → Detection → AI Analysis → n8n Webhook → Slack Alerts

---

## 🔧 Technologies Used

* Python (Detection + AI Analysis)
* n8n (Workflow Automation)
* Slack Webhooks (Alerting)

---

## ⚡ Features

* Real-time alert automation
* AI-enriched threat intelligence
* Structured JSON-based workflow
* Scalable and modular design

---

## 📡 Workflow

1. Python script detects attack
2. AI analyzes threat
3. JSON sent to n8n webhook
4. n8n processes alert
5. Slack notification triggered
